# Лабораторная работа 7

В этой работе используется база данных SQLite и простой текстовый интерфейс (TUI).

## Структура

- `db.py` — работа с файлом базы данных, создание подключения, начальная инициализация.
- `create_tables.py` — создание нескольких таблиц (`departments`, `employees`).
- `insert_data.py` — вставка тестовых данных в таблицы.
- `update_data.py` — обновление записи (зарплата сотрудника по его `id`).
- `tui.py` — текстовый интерфейс, из которого можно вызывать все операции.

## Как запустить

Из корня проекта:

```bash
python -m lr7.tui
```

Доступные действия в меню:

1. Создать таблицы.
2. Вставить тестовые данные.
3. Обновить запись (изменить зарплату сотрудника по `id`).
4. Посмотреть содержимое таблиц `departments` и `employees`.

База данных хранится в файле `lr7/db.db` рядом с модулем `db.py`.
