# Лабораторная работа 5: Графические интерфейсы (GUI) на PyQt5

## Описание

Данная лабораторная работа демонстрирует создание графических интерфейсов с использованием PyQt5. Изучаются основные компоненты GUI, работа с виджетами, меню, рисование и создание интерактивных приложений.

## Структура проекта

```
lr5/
├── text_input_window.py    # Окно с текстовым полем и кнопкой
├── listbox_window.py       # Список (Listbox) и выбор элемента
├── menu_window.py          # Окно с меню (File→Exit)
├── drawing_window.py       # Рисование фигур мышью
├── currency_converter.py   # Конвертер валют
├── main.py                 # Главное меню для запуска всех приложений
├── requirements.txt        # Зависимости проекта
└── README.md               # Документация
```

## Выполненные задания

### 1. Окно с текстовым полем и кнопкой, выводящей текст

Окно с полем ввода текста и областью вывода.

**Файл:** `text_input_window.py`

**Запуск:**
```bash
python text_input_window.py
```

**Функциональность:**
- QLineEdit для ввода текста
- QTextEdit для вывода текста (только для чтения)
- Кнопка "Вывести текст" для переноса текста из поля ввода в область вывода
- Кнопка "Очистить" для очистки области вывода
- Поддержка Enter для отправки текста

**Использование:**
1. Введите текст в поле ввода
2. Нажмите кнопку "Вывести текст" или Enter
3. Текст появится в области вывода с префиксом "Вы ввели:"

### 2. Список (Listbox) и выбор элемента

Приложение со списком элементов и возможностью выбора.

**Файл:** `listbox_window.py`

**Запуск:**
```bash
python listbox_window.py
```

**Функциональность:**
- QListWidget для отображения списка элементов
- Выбор элемента одинарным или двойным кликом
- Отображение выбранного элемента
- Добавление новых элементов в список
- Удаление выбранных элементов
- Диалоговые окна для подтверждения действий

**Использование:**
1. Выберите элемент из списка одинарным кликом
2. Двойной клик или кнопка "Показать выбранный элемент" покажет выбранный элемент
3. Используйте кнопки для добавления/удаления элементов

### 3. Окно с меню (File→Exit)

Окно с меню-баром и пунктом меню File→Exit.

**Файл:** `menu_window.py`

**Запуск:**
```bash
python menu_window.py
```

**Функциональность:**
- QMenuBar для создания меню
- Меню File с пунктами:
  - New (Ctrl+N)
  - Open (Ctrl+O)
  - Exit (Ctrl+Q)
- Меню Help с пунктом About
- Горячие клавиши для быстрого доступа
- Подтверждение выхода из приложения

**Использование:**
1. Используйте меню File для доступа к функциям
2. File→Exit или Ctrl+Q для выхода
3. Help→About для информации о программе

### 4. Рисование фигур мышью

Приложение для рисования фигур на холсте с помощью мыши.

**Файл:** `drawing_window.py`

**Запуск:**
```bash
python drawing_window.py
```

**Функциональность:**
- Выбор типа фигуры (прямоугольник, круг, линия)
- Рисование фигур мышью (зажать и перетащить)
- Отображение фигур в реальном времени
- Сохранение всех нарисованных фигур
- Кнопка очистки холста
- Белый холст с рамкой для рисования

**Использование:**
1. Выберите тип фигуры (Прямоугольник, Круг или Линия)
2. Зажмите левую кнопку мыши на холсте
3. Перетащите мышь для создания фигуры
4. Отпустите кнопку мыши для завершения
5. Используйте кнопку "Очистить" для удаления всех фигур

### 5. Программа «Конвертер валют»

Приложение для конвертации валют.

**Файл:** `currency_converter.py`

**Запуск:**
```bash
python currency_converter.py
```

**Функциональность:**
- Поле ввода суммы
- Выбор исходной валюты (USD, EUR, GBP, JPY, CNY, RUB)
- Выбор целевой валюты
- Конвертация через рубли
- Отображение результата
- Информация о курсах валют
- Обработка ошибок ввода

**Поддерживаемые валюты:**
- USD (доллар США)
- EUR (евро)
- GBP (фунт стерлингов)
- JPY (японская иена)
- CNY (китайский юань)
- RUB (российский рубль)

**Использование:**
1. Введите сумму для конвертации
2. Выберите исходную валюту
3. Выберите целевую валюту
4. Нажмите "Конвертировать"

## Основные компоненты PyQt5

### QMainWindow
Основное окно приложения с меню, панелями инструментов и статусной строкой.

### QWidget
Базовый класс для всех виджетов. Используется как контейнер для других элементов.

### QLabel
Виджет для отображения текста или изображений.

### QPushButton
Кнопка, которая может быть нажата пользователем.

### QLineEdit
Однострочное текстовое поле для ввода текста.

### QTextEdit
Многострочное текстовое поле для ввода и отображения текста.

### QListWidget
Виджет списка для отображения и выбора элементов.

### QComboBox
Выпадающий список для выбора одного элемента из списка.

### QMenuBar, QMenu, QAction
Компоненты для создания меню приложения.

### QPainter
Класс для рисования графики на виджетах.

### Layouts (QVBoxLayout, QHBoxLayout)
Менеджеры компоновки для автоматического размещения виджетов.
