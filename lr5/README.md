# Лабораторная работа 5: Графические интерфейсы (GUI) на PyQt5

## Описание

Данная лабораторная работа демонстрирует создание графических интерфейсов с использованием PyQt5. Изучаются основные компоненты GUI, работа с виджетами, диалогами и создание интерактивных приложений.

## Структура проекта

```
lr5/
├── simple_gui.py           # Простейший GUI на PyQt5
├── text_input_window.py     # Окно с текстовым полем и кнопкой
├── file_dialog.py          # Диалог выбора файла
├── local_chat.py           # Локальный чат
├── clicker_game.py         # Игра «Кликер»
├── main.py                 # Главное меню для запуска всех приложений
└── README.md               # Документация
```

## Выполненные задания

### 1. Простейший GUI на PyQt5

Простое окно с приветствием и кнопкой закрытия.

**Файл:** `simple_gui.py`

**Запуск:**
```bash
python simple_gui.py
```

**Особенности:**
- Использование QMainWindow как основного окна
- QLabel для отображения текста
- QPushButton для кнопки закрытия
- QVBoxLayout для вертикального размещения элементов

### 2. Окно с текстовым полем и кнопкой, выводящей текст

Окно с полем ввода текста и областью вывода.

**Файл:** `text_input_window.py`

**Запуск:**
```bash
python text_input_window.py
```

**Функциональность:**
- QLineEdit для ввода текста
- QTextEdit для вывода текста (только для чтения)
- Кнопка "Вывести текст" для переноса текста из поля ввода в область вывода
- Кнопка "Очистить" для очистки области вывода
- Автоматическая очистка поля ввода после отправки

**Использование:**
1. Введите текст в поле ввода
2. Нажмите кнопку "Вывести текст" или Enter
3. Текст появится в области вывода с префиксом "Вы ввели:"

### 3. Диалог выбора файла

Приложение с диалогами открытия и сохранения файлов.

**Файл:** `file_dialog.py`

**Запуск:**
```bash
python file_dialog.py
```

**Функциональность:**
- QFileDialog.getOpenFileName() для выбора файла для открытия
- QFileDialog.getSaveFileName() для выбора пути сохранения
- Чтение и отображение содержимого текстовых файлов
- Сохранение файла с примерным содержимым
- Фильтры файлов (все файлы, текстовые, Python)

**Использование:**
1. Нажмите "Выбрать файл" для открытия диалога выбора файла
2. Выберите файл - его содержимое отобразится в текстовом поле
3. Нажмите "Сохранить файл" для сохранения нового файла

### 4. Программа чат (локальный)

Локальный чат с возможностью отправки сообщений.

**Файл:** `local_chat.py`

**Запуск:**
```bash
python local_chat.py
```

**Функциональность:**
- Поле для ввода имени пользователя
- Область отображения сообщений (только для чтения)
- Поле ввода сообщения
- Кнопка отправки сообщения
- Автоматическое добавление временной метки к каждому сообщению
- Кнопка очистки чата
- Автопрокрутка к последнему сообщению

**Использование:**
1. Введите ваше имя (по умолчанию "Пользователь")
2. Введите сообщение в поле ввода
3. Нажмите "Отправить" или Enter
4. Сообщение появится в области чата с временной меткой

### 5. Игра «Кликер»

Игра-кликер с системой улучшений.

**Файл:** `clicker_game.py`

**Запуск:**
```bash
python clicker_game.py
```

**Функциональность:**
- Кнопка для кликов, увеличивающая счет
- Система автокликера, который автоматически увеличивает счет
- Улучшение автокликера за очки
- Отображение текущего счета
- Отображение уровня автокликера
- Кнопка сброса игры
- Визуальное оформление с использованием стилей

**Механика игры:**
- Каждый клик по кнопке увеличивает счет на 1
- Автокликер автоматически добавляет очки каждую секунду
- Улучшение автокликера стоит: (уровень + 1) * 10 очков
- Каждое улучшение увеличивает скорость автокликера на 1 клик/сек

**Использование:**
1. Кликайте по большой зеленой кнопке для увеличения счета
2. Когда накопите достаточно очков, улучшайте автокликер
3. Автокликер будет автоматически добавлять очки
4. Используйте кнопку "Сбросить игру" для начала заново

## Основные компоненты PyQt5

### QMainWindow
Основное окно приложения с меню, панелями инструментов и статусной строкой.

### QWidget
Базовый класс для всех виджетов. Используется как контейнер для других элементов.

### QLabel
Виджет для отображения текста или изображений.

### QPushButton
Кнопка, которая может быть нажата пользователем.

### QLineEdit
Однострочное текстовое поле для ввода текста.

### QTextEdit
Многострочное текстовое поле для ввода и отображения текста.

### QFileDialog
Диалог для выбора файлов (открытие и сохранение).

### Layouts (QVBoxLayout, QHBoxLayout)
Менеджеры компоновки для автоматического размещения виджетов.
