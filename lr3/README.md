# LR3 (Python OOP)

В папке реализованы задания:

1. **Класс "Студент"** с методом вывода информации — см. `student.py`.
2. **Класс "Фигура"** с методами вычисления площади и периметра + реализации (Rectangle/Circle/Triangle) — см. `figure.py`.
3. **Перегрузка оператора `+`** — пример на `Vector2D` — см. `operator_overload.py`.
4. **Паттерн Factory** — `FigureFactory`, который создаёт фигуры по строковому ключу — см. `factory.py`.
5. **Наследование от встроенного `list`** + дополнительные методы (`unique`, `flatten`, `average`) — см. `custom_list.py`.

## Запуск примеров

Из папки проекта:

```bash
python3 lr3/main.py
```

Либо можно запускать любой файл отдельно:

```bash
python3 lr3/student.py
python3 lr3/figure.py
```
