# LR2 (Python basics)

В папке реализованы задания:

1. **Сумма всех нечётных чисел до N** — функция [`sum_odd_up_to()`](lr2/sum_odd.py:6).
2. **НОД двух чисел** — функция [`gcd()`](lr2/gcd_calc.py:6).
3. **Подсчёт гласных** (русские + английские буквы) — функция [`count_vowels()`](lr2/vowel_count.py:11).

Каждый файл можно запускать напрямую, в нём есть простой ввод данных через `input()` и вывод результата на экран.
